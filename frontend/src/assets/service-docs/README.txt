Upload service-related PDFs in this folder.

Example:
- internet-access-academic.pdf
- internet-access-residential.pdf
- wifi-android.pdf
- wifi-windows10.pdf

The service pages will automatically use the uploaded local file when the
filename matches the `assetName` field in `src/data/services.js`.
If a file is not present yet, the page falls back to the official CIC link.
